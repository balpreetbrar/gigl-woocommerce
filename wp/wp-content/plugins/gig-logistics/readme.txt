GIG Logistics adds GIG Logistics shipping, rate calculation, and tracking to WooCommerce checkout.

== Description ==

GIGL Delivery integrates GIG Logistics (GIGL) shipping into WooCommerce so merchants can offer real-time shipping rates, capture shipments, and provide tracking to customers. Features include cash-on-delivery support, live/test mode, sender address management, weight-based calculations, and order scheduling.

== Installation ==

= Automatic installation =

1. Go to Plugins → Add New in your WordPress admin.
2. Search for "GIG Logistics" and click "Install Now".
3. Activate the plugin and configure settings under WooCommerce → Settings → Shipping or GIGL Logistics → GIG Settings.

= Manual installation =

1. Upload the `gig-logistics` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins screen in WordPress.
3. Configure settings under WooCommerce → Settings → Shipping or GIGL Logistics → GIG Settings.

== Frequently Asked Questions ==

= What do I need to use the plugin? =

* WooCommerce must be installed and activated.
* A GIGL merchant account is required (https://giglogistics.com/sign-in).
